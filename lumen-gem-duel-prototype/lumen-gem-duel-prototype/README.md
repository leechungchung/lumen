# Lumen Gem Duel Prototype

2인 로컬 플레이용 모바일 웹 MVP 프로토타입입니다.

## 포함 기능

- 카드 시장 1/2/3단계
- 보석 5종, S/A/B/C/D 등급 가치
- 스플렌더식 카드 구매와 할인 누적
- 구매 순간 미니게임 발생
- 획득자 vs 방해자 점수 차이로 카드 등급 결정
- 방해자 파편 보석 보상
- S등급 랜덤 각인
- 명성 15점 승리

## 실행

```bash
npm install
npm run dev
```

브라우저에서 표시되는 로컬 주소로 접속하세요.

## 참고

현재 버전은 실시간 서버가 없는 로컬 2인 프로토타입입니다. Firebase/Supabase 방 코드 실시간 대전은 다음 단계에서 붙이면 됩니다.
