# 루멘 젬 듀얼 온라인

Firebase Realtime Database 기반 2인 모바일 웹게임입니다. 카드 구매로 엔진을 만들고, 구매 직후 공개되는 실시간 세공 미니게임 결과로 `대성공 / 성공 / 패배` 보상을 받습니다.

## 이번 버전 핵심

- 온라인 방 만들기/참가, player1/player2 역할 잠금, 초대 코드/초대문 복사
- 카드 이름표와 광물색 기반 시너지 족보
- 세공시장 카드 테두리에 대표 시너지 색상 표시
- 세공시장 카드에 `시너지 발동`, `상대 발동 위험`, `조합 진척` 표시
- 미니게임 사전 노출 제거: 구매 후 5초 데모와 함께 게임 공개
- `assets/duel-arena.png` 아레나 배경 추가로 미니게임 화면 품질 강화
- 대성공 기준 강화: 큰 점수 차이와 충분한 세공 점수를 동시에 만족해야 하며, S급 특별 능력은 대성공에서만 부여
- 포트리스 세공포 개편: 시간 경쟁이 아니라 곡사 포물선으로 핵 체력을 깎는 구조
- 자력 광물 운반 개편: 세공자는 목적지 N/S극, 방해자는 광물 N/S극을 바꾸며 방해자는 연속 광클 시 과부하
- 프리즘 승부차기 개편: 왼쪽/중앙/오른쪽 방향 읽기와 역할 교대가 명확한 구조
- 낚시 게임 추가: 한 번 던진 찌로 지나가는 물고기를 많이 잡는 게임
- Service Worker 네트워크 우선 캐시로 배포 후 구버전 캐시 문제 완화

## 시너지 족보

- 왕실 세공단: `왕실` 이름표. 성공 이상 명성 보너스, 대성공 추가 명성
- 오색 프리즘: 서로 다른 광물색. 모든 구매 비용 감소, 대성공 별가루
- 용광로 연맹: `화염/장치`. 용광로와 포트리스 강화, 광맥 보너스
- 별빛 합창단: `별빛/공명`. 리듬 판정 완화, 술래잡기 강화
- 생명 균형단: `생장/균형`. 방해 점수 감소, 줄다리기/광물 운반 강화
- 황금 교역로: `황금/교역`. 광맥 수급과 성공 보너스 강화
- 거울 예지회: `거울/예지`. 색상 연쇄와 승부차기 강화
- 결투장 맹세: `결투`. 에어하키, 승부차기, 술래잡기 강화

## Firebase 설정

`firebase-config.js`에 Firebase Web App 설정값을 넣습니다. Realtime Database 테스트 규칙 예시는 아래와 같습니다.

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

출시 전에는 반드시 인증 기반 규칙으로 바꾸는 것을 권장합니다.

## Vercel 배포

- Framework Preset: Other
- Build Command: 비워두기
- Output Directory: `.`
- Root Directory: `index.html`이 들어있는 폴더

정적 파일만으로 동작하므로 별도 빌드 과정은 없습니다.
